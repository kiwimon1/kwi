## 1. Networking (2.0점)

### 1-1. VPC & Subnet (1.0점)

**구성 내용:**
- VPC: `wsc2026-skills-vpc` (192.168.0.0/16)
- Public Subnet: hub-sub-a (192.168.1.0/24, AZ-a), hub-sub-b (192.168.10.0/24, AZ-b)
- Private Subnet: app-sub-a (192.168.2.0/24, AZ-a), app-sub-b (192.168.20.0/24, AZ-b)

```bash
# VPC 생성
VPC_ID=$(aws ec2 create-vpc --cidr-block 192.168.0.0/16 --tag-specifications 'ResourceType=vpc,Tags=[{Key=Name,Value=wsc2026-skills-vpc}]' --query "Vpc.VpcId" --output text)
aws ec2 modify-vpc-attribute --vpc-id $VPC_ID --enable-dns-hostnames
aws ec2 modify-vpc-attribute --vpc-id $VPC_ID --enable-dns-support

# 서브넷 생성
SUB_HUB_A=$(aws ec2 create-subnet --vpc-id $VPC_ID --cidr-block 192.168.1.0/24 --availability-zone ap-northeast-2a --tag-specifications 'ResourceType=subnet,Tags=[{Key=Name,Value=wsc2026-skills-hub-sub-a}]' --query "Subnet.SubnetId" --output text)
SUB_HUB_B=$(aws ec2 create-subnet --vpc-id $VPC_ID --cidr-block 192.168.10.0/24 --availability-zone ap-northeast-2b --tag-specifications 'ResourceType=subnet,Tags=[{Key=Name,Value=wsc2026-skills-hub-sub-b}]' --query "Subnet.SubnetId" --output text)
SUB_APP_A=$(aws ec2 create-subnet --vpc-id $VPC_ID --cidr-block 192.168.2.0/24 --availability-zone ap-northeast-2a --tag-specifications 'ResourceType=subnet,Tags=[{Key=Name,Value=wsc2026-skills-app-sub-a}]' --query "Subnet.SubnetId" --output text)
SUB_APP_B=$(aws ec2 create-subnet --vpc-id $VPC_ID --cidr-block 192.168.20.0/24 --availability-zone ap-northeast-2b --tag-specifications 'ResourceType=subnet,Tags=[{Key=Name,Value=wsc2026-skills-app-sub-b}]' --query "Subnet.SubnetId" --output text)

# Public 서브넷 자동 퍼블릭 IP 할당
aws ec2 modify-subnet-attribute --subnet-id $SUB_HUB_A --map-public-ip-on-launch
aws ec2 modify-subnet-attribute --subnet-id $SUB_HUB_B --map-public-ip-on-launch
```

### 1-2. Routing - IGW, NAT, RTB (1.0점)

**구성 내용:**
- IGW → hub-rtb (Public)
- NAT-A → app-rtb-a, NAT-B → app-rtb-b (Private, AZ별 NAT)

```bash
# Internet Gateway
IGW_ID=$(aws ec2 create-internet-gateway --tag-specifications 'ResourceType=internet-gateway,Tags=[{Key=Name,Value=wsc2026-skills-igw}]' --query "InternetGateway.InternetGatewayId" --output text)
aws ec2 attach-internet-gateway --internet-gateway-id $IGW_ID --vpc-id $VPC_ID

# Elastic IP for NAT
EIP_A=$(aws ec2 allocate-address --query "AllocationId" --output text)
EIP_B=$(aws ec2 allocate-address --query "AllocationId" --output text)

# NAT Gateway (각 Public 서브넷에 배치)
NAT_A=$(aws ec2 create-nat-gateway --subnet-id $SUB_HUB_A --allocation-id $EIP_A --tag-specifications 'ResourceType=natgateway,Tags=[{Key=Name,Value=wsc2026-skills-nat-a}]' --query "NatGateway.NatGatewayId" --output text)
NAT_B=$(aws ec2 create-nat-gateway --subnet-id $SUB_HUB_B --allocation-id $EIP_B --tag-specifications 'ResourceType=natgateway,Tags=[{Key=Name,Value=wsc2026-skills-nat-b}]' --query "NatGateway.NatGatewayId" --output text)

# NAT가 available 될 때까지 대기
aws ec2 wait nat-gateway-available --nat-gateway-ids $NAT_A $NAT_B

# Route Table - Public (hub)
HUB_RTB=$(aws ec2 create-route-table --vpc-id $VPC_ID --tag-specifications 'ResourceType=route-table,Tags=[{Key=Name,Value=wsc2026-skills-hub-rtb}]' --query "RouteTable.RouteTableId" --output text)
aws ec2 create-route --route-table-id $HUB_RTB --destination-cidr-block 0.0.0.0/0 --gateway-id $IGW_ID
aws ec2 associate-route-table --route-table-id $HUB_RTB --subnet-id $SUB_HUB_A
aws ec2 associate-route-table --route-table-id $HUB_RTB --subnet-id $SUB_HUB_B

# Route Table - Private A
APP_RTB_A=$(aws ec2 create-route-table --vpc-id $VPC_ID --tag-specifications 'ResourceType=route-table,Tags=[{Key=Name,Value=wsc2026-skills-app-rtb-a}]' --query "RouteTable.RouteTableId" --output text)
aws ec2 create-route --route-table-id $APP_RTB_A --destination-cidr-block 0.0.0.0/0 --nat-gateway-id $NAT_A
aws ec2 associate-route-table --route-table-id $APP_RTB_A --subnet-id $SUB_APP_A

# Route Table - Private B
APP_RTB_B=$(aws ec2 create-route-table --vpc-id $VPC_ID --tag-specifications 'ResourceType=route-table,Tags=[{Key=Name,Value=wsc2026-skills-app-rtb-b}]' --query "RouteTable.RouteTableId" --output text)
aws ec2 create-route --route-table-id $APP_RTB_B --destination-cidr-block 0.0.0.0/0 --nat-gateway-id $NAT_B
aws ec2 associate-route-table --route-table-id $APP_RTB_B --subnet-id $SUB_APP_B
```

## 2. Database - DynamoDB (1.3점)

### 2-1. DynamoDB Table (1.3점)

**구성 내용:**
- Table: wsc2026-book-table, PK: client_id, PAY_PER_REQUEST
- KMS CMK 암호화 (wsc2026-db-kms)
- 삭제 방지 활성화, PITR 35일, GSI (booking_id)
- Resource Policy: Pod에 PutItem, Lambda에 Query 권한

```bash
# KMS Key 생성 (최소 권한 - root/kms:* 금지)
DB_KMS_ID=$(aws kms create-key --description "wsc2026-db-kms" --policy '{
  "Version": "2012-10-17",
  "Statement": [
    {
      "Sid": "AllowKeyAdministration",
      "Effect": "Allow",
      "Principal": {"AWS": "arn:aws:iam::'$ACCOUNT_ID':role/Admin"},
      "Action": [
        "kms:Create*","kms:Describe*","kms:Enable*","kms:List*","kms:Put*",
        "kms:Update*","kms:Revoke*","kms:Disable*","kms:Get*","kms:Delete*",
        "kms:TagResource","kms:UntagResource","kms:ScheduleKeyDeletion","kms:CancelKeyDeletion"
      ],
      "Resource": "*"
    },
    {
      "Sid": "AllowKeyUsage",
      "Effect": "Allow",
      "Principal": {"AWS": "arn:aws:iam::'$ACCOUNT_ID':role/Admin"},
      "Action": ["kms:Decrypt","kms:DescribeKey","kms:Encrypt","kms:ReEncrypt*","kms:GenerateDataKey*"],
      "Resource": "*"
    },
    {
      "Sid": "AllowDynamoDBUsage",
      "Effect": "Allow",
      "Principal": {"Service": "dynamodb.amazonaws.com"},
      "Action": ["kms:Decrypt","kms:DescribeKey","kms:Encrypt","kms:ReEncrypt*","kms:GenerateDataKey*"],
      "Resource": "*"
    }
  ]
}' --query "KeyMetadata.KeyId" --output text)

aws kms create-alias --alias-name alias/wsc2026-db-kms --target-key-id $DB_KMS_ID

# DynamoDB 테이블 생성
aws dynamodb create-table \
  --table-name wsc2026-book-table \
  --attribute-definitions AttributeName=client_id,AttributeType=S AttributeName=booking_id,AttributeType=S \
  --key-schema AttributeName=client_id,KeyType=HASH \
  --billing-mode PAY_PER_REQUEST \
  --sse-specification Enabled=true,SSEType=KMS,KMSMasterKeyId=$DB_KMS_ID \
  --deletion-protection-enabled \
  --global-secondary-indexes '[{
    "IndexName": "booking_id-index",
    "KeySchema": [{"AttributeName": "booking_id", "KeyType": "HASH"}],
    "Projection": {"ProjectionType": "ALL"}
  }]'

# 테이블 생성 대기
aws dynamodb wait table-exists --table-name wsc2026-book-table

# PITR 활성화 (최장 기간 = 35일)
aws dynamodb update-continuous-backups \
  --table-name wsc2026-book-table \
  --point-in-time-recovery-specification PointInTimeRecoveryEnabled=true,RecoveryPeriodInDays=35

# Resource Policy (Pod: PutItem, Lambda: Query)
aws dynamodb put-resource-policy --resource-arn "arn:aws:dynamodb:ap-northeast-2:${ACCOUNT_ID}:table/wsc2026-book-table" --policy '{
  "Version": "2012-10-17",
  "Statement": [
    {
      "Effect": "Allow",
      "Principal": {"AWS": "arn:aws:iam::'$ACCOUNT_ID':role/wsc2026-book-pod-role"},
      "Action": "dynamodb:PutItem",
      "Resource": "arn:aws:dynamodb:ap-northeast-2:'$ACCOUNT_ID':table/wsc2026-book-table"
    },
    {
      "Effect": "Allow",
      "Principal": {"AWS": "arn:aws:iam::'$ACCOUNT_ID':role/wsc2026-book-function-role"},
      "Action": "dynamodb:Query",
      "Resource": "arn:aws:dynamodb:ap-northeast-2:'$ACCOUNT_ID':table/wsc2026-book-table*"
    }
  ]
}'
```

from boto3.dynamodb.conditions import Key

table_name = os.environ['TABLE_NAME']
table = dynamodb.Table(table_name)

def lambda_handler(event, context):
    booking_id = event.get('queryStringParameters', {}).get('booking_id', '')
    if not booking_id:
        return {'statusCode': 400, 'body': json.dumps({'error': 'booking_id required'})}
    
    response = table.query(
        IndexName='booking_id-index',
        KeyConditionExpression=Key('booking_id').eq(booking_id)
    )
    
    items = response.get('Items', [])
    if not items:
        return {'statusCode': 404, 'body': json.dumps({'error': 'not found'})}
    
    item = items[0]
    result = {

        'email': item.get('email', ''),
        'concert_name': item.get('concert_name', ''),

    }
